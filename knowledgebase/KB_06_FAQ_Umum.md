# FAQ — Pertanyaan Umum

## Tentang Produk

**Q: Apa itu ModalHP?**
A: Aplikasi web (PWA) berisi 15 tools bisnis UMKM dalam satu platform. Bayar sekali, pakai seumur hidup.

**Q: ModalHP itu aplikasi atau website?**
A: Keduanya! PWA bisa diakses lewat browser dan bisa di-install jadi aplikasi di HP tanpa Play Store/App Store.

**Q: Bisa dipakai di HP kentang/spesifikasi rendah?**
A: Bisa! Dioptimalkan untuk berjalan cepat di semua jenis HP, termasuk RAM kecil.

**Q: Harus pakai internet?**
A: Tidak. Setelah install, bisa digunakan 100% offline. Data tetap tersimpan aman di HP.

## Data & Privasi

**Q: Data disimpan di mana?**
A: 100% lokal di perangkat Anda (IndexedDB). Kami tidak menyimpan/mengirim data ke server manapun.

**Q: Apakah data aman?**
A: Sangat aman. Data lokal di perangkat, tidak ada pihak lain yang bisa akses — termasuk kami.

**Q: Kalau HP hilang/rusak?**
A: Data di perangkat tersebut tidak bisa dipulihkan. Sarankan rutin ekspor/backup data jika tersedia fitur ekspornya.

**Q: Data bisa dipindah ke HP baru?**
A: Data tidak otomatis berpindah (offline-first). Gunakan fitur ekspor data untuk pindah manual.

## Pembayaran

**Q: Ada biaya bulanan?**
A: TIDAK. Bayar sekali, pakai selamanya. Tanpa hidden fee.

**Q: Ada garansi uang kembali?**
A: Ya! Garansi 14 hari penuh. Tidak cocok? Uang dikembalikan.

**Q: Ada biaya tambahan untuk fitur baru?**
A: Tidak. Semua update fitur masa depan GRATIS untuk yang sudah membeli.
