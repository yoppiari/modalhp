# Panduan Instalasi & Akses ModalHP

## Cara Mengakses
1. Buka browser (Chrome disarankan) di HP atau laptop.
2. Kunjungi **modalhp.com**.
3. Klik tombol **"Login Sekarang"** atau **"Daftar Sekarang"**.
4. Buat akun atau login.

## Install di HP Android (Chrome)
1. Buka `modalhp.com` di Google Chrome.
2. Ketuk ikon **"⋮" (titik tiga)** di pojok kanan atas.
3. Pilih **"Tambahkan ke layar utama"** / **"Install app"**.
4. Ketuk **"Install"** / **"Tambahkan"**.
5. Ikon ModalHP muncul di layar utama HP seperti aplikasi biasa.

## Install di iPhone (Safari)
1. Buka `modalhp.com` di **Safari** (wajib Safari, bukan Chrome).
2. Ketuk ikon **Share (kotak dengan panah ke atas)** di bawah.
3. Scroll ke bawah, pilih **"Tambahkan ke Layar Utama"**.
4. Ketuk **"Tambah"**.

## Install di Laptop/PC (Chrome)
1. Buka `modalhp.com` di Chrome.
2. Klik ikon **install (⊕)** di address bar.
3. Klik **"Install"**.

## Setelah Install
- Aplikasi bisa dipakai **offline** (tanpa internet).
- Data tersimpan di perangkat masing-masing.
- Untuk update: cukup buka aplikasi saat ada internet → update otomatis.

## Catatan Penting
- Login & Daftar akun **membutuhkan internet**.
- Setelah login, semua fitur bisa dipakai offline.
- Data tidak otomatis berpindah antar perangkat (karena disimpan lokal).
