# Troubleshooting — Masalah Umum

## 🔴 Layar putih / blank screen
**Penyebab:** Cache Service Worker lama bentrok update.
**Solusi:**
1. Chrome → Settings → Privacy → Clear Browsing Data
2. Centang "Cached images and files" & "Cookies and other site data"
3. Clear Data → buka ulang modalhp.com → install ulang PWA
4. Jika masih error, coba buka di Incognito/Private mode

## 🔴 Tidak bisa install aplikasi di HP
**Solusi:**
- Android: wajib pakai **Google Chrome**
- iPhone: wajib pakai **Safari**
- Browser lain (Firefox, Samsung Internet) mungkin tidak support PWA install
- Pastikan sudah akses modalhp.com (bukan URL lain)

## 🔴 Data transaksi hilang
**Penyebab:** Pelanggan hapus cache/data browser → data lokal ikut terhapus, atau ganti HP tanpa backup.
**Solusi:**
- Jelaskan data disimpan lokal di perangkat
- **JANGAN pernah** hapus data browser untuk domain modalhp.com
- Sarankan rutin ekspor data sebagai backup

## 🔴 Fitur tidak muncul / tampilan rusak
**Solusi:**
1. Refresh halaman (tarik ke bawah / F5)
2. Jika tidak berhasil, clear cache dan buka ulang
3. Pastikan browser versi terbaru
4. Masih rusak → eskalasi ke tim teknis

## 🔴 Struk tidak bisa di-share ke WhatsApp
**Solusi:**
- Pastikan WhatsApp terinstall
- Pastikan browser punya izin (permission) sharing
- Restart HP lalu buka ulang ModalHP

## 🔴 Login gagal / tidak bisa daftar
**Solusi:**
1. Pastikan internet aktif (login/daftar butuh internet)
2. Pastikan email belum terdaftar sebelumnya
3. Coba browser lain atau incognito mode
4. Masih gagal → eskalasi ke tim teknis + sertakan screenshot
